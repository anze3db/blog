At my day job we have a Django app with almost `500_000` lines of Python code that was written over the last decade. Ever since we migrated to Python 3.7 (almost seven months *after* Python 2 EOL ðŸ˜“), we've tried to keep on top on Python upgrades. We migrated to 3.9 at the end of last year, skipping 3.8 completely. It was now time for us to jump on Python 3.10.
